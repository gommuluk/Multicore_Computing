-README: prob2 (MatmultD)-

이 코드는 이클립스 또는 cmd창을 통해 실행할 수 있습니다. 

  *  cmd 실행 : cygwin에서 "terminal> java MatmultD 6(스레드 개수) < mat500.txt(사용할 매트릭스 파일)"
  *  이클립스 실행 : 상단 메뉴에서 [Run]->[Run Configurations]-> [Arguments] -> [Program arguments]에 스레드 개수를 넣고, [Run]->[Run Configurations]-> [Common] -> [Standart Input and Output] -> [Input File]에서 사용할 매트릭스 파일을 설정합니다. 

매트릭스 파일은 곱셈연산을 할 두 개의 매트릭스 정보를 담고 있습니다.
