-README: prob1(dynamic)-

이 코드는 이클립스 또는 cmd창을 통해 실행할 수 있습니다. 
별도의 설정 값은 없습니다 .

메인함수 바로 위쪽에 선언된 변수를 수정하여 다른 범위에 대한 스레드 성능을 확인할 수 있습니다. 
  *  private static final int NUM_END : 소수의 개수를 알아보고 싶은 숫자 범위 가장 마지막 수 입니다. 
  *  private static final int NUM_THREAD : 스레드의 개수입니다. 
  *  private static final int NUM_TASK : 전체 범위를 NUM_TASK개로 쪼개서 일을 처리합니다.
